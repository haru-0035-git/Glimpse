# Glimpse Frontend (Vite)

## Available Scripts

### `npm run dev`

Starts the Vite dev server (default port 3001, configured in `vite.config.ts`).

### `npm run build`

Builds the app for production into the `dist` folder.

### `npm run preview`

Locally previews the production build.

## Notes

- Dev API proxy: `/api` -> `http://localhost:8080` (see `vite.config.ts`).
